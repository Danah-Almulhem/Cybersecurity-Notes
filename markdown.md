
# Markdown Basics

| Symbol | Purpose |
|--------|---------|
| # | Creates a level 1 heading (largest heading). |
| ## | Creates a level 2 heading. |
| ### | Creates a level 3 heading. |
| - or * | Creates a bullet point (unordered list). |
| 1. | Creates a numbered list. |
| ` `text `` | Formats text as inline code (used for commands like `pwd`). |
|  | Creates a code block for multiple lines of code. |
| **text** | Makes text bold. |
| *text* or _text_ | Makes text *italic*. |
| > | Creates a block quote. |
| [Text](URL) | Creates a hyperlink. |
| --- | Creates a horizontal line (divider). |
| | | Used to create tables. |
