
#Networking

##OSI Model

1. Physical 
- converts data into bits to carry it across physical network equpiment.

2. Data link
- Handles node to node data  transfer brtween devices on the same network.
#Protocols
- Ethernet : The standard protocol for LANs and it relies on MAC addressing to identify physical hardware
- WIFI : the standard protocol that governing WLANs. it manages media access and prevent signal collosion with CSMA/CA

3. Network
- conducts routing between different networks, determining the most efficient path.

4. Transport
- handles data transmission between devices using UDP and TCP protocols.

5. Session 
- manages connections and the opening and closing of sessions between devices.

6. Presentation
Configures data into an acceptable format via translation,encryption, and compression.

7. Application
- Accesses protocols used by applications to communicate data to end users
