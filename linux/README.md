# Linux

## What is Linux?

Linux is a free and open-source operating system used in servers, cloud computing, cybersecurity, and software development. It is known for its stability, security, and flexibility.

---

## Why is Linux Important in Cybersecurity?

Linux is one of the most important operating systems in cybersecurity because:

- Most servers run Linux.
- Many cybersecurity tools are built for Linux.
- It provides powerful command-line utilities.
- It allows better control over the operating system.
- Kali Linux is based on Linux.

---

## Linux File System

Linux uses a hierarchical file system that starts from the root directory (`/`).

Some important directories:

| Directory | Description |
|-----------|-------------|
| / | Root directory |
| /home | User home directories |
| /etc | System configuration files |
| /bin | Essential commands |
| /usr | User applications |
| /var | Log and variable files |
| /tmp | Temporary files |

---

## Linux Terminal

The terminal is a command-line interface (CLI) that allows users to interact with the operating system by typing commands.
